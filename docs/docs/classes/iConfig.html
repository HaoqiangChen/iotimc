    iConfig Class

    系统配置服务。获取系统配置，包括主标题、副标题、logo等。

    1. 主框架已加载，无需再次引入。
    2. 此类为服务，需要依赖注入才能使用。
     JS:
 app.controller('***Controller', ['iConfig', function(iConfig) {
     // TODO
 }]);
