    iGetLang Class

    方言管理服务。方言管理主要应用于不同行业的名称通过数据库进行配置，而无需写死代码中，例如监狱中的罪犯，在戒毒中就是学员。

    1. 主框架已加载，无需再次引入。
    2. 此类为服务，需要依赖注入才能使用。
     JS:
 app.controller('***Controller', ['iGetLang', function(iGetLang) {
     // TODO
 }]);

    根据key，返回目前数据库对应的方言，如当前数据库配置的行业为监狱，那么返回对被管理人员的称呼为罪犯；

    Parameters:

    nametypeflagdescription
    key String

    需要获取方言查询的key，参数具体根据数据库情况配置。

    Returns:

    [String]

    返回key对应的方言。

    Example:

     // 注入iGetLang服务。
 app.controller('***Controller', ['iGetLang', function(iGetLang) {
     console.log(iGetLang.get('CPN'));    // 罪犯
 }]);
