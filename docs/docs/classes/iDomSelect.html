    iDomSelect Class

    当焦点在输入框时允许选择，离开时禁止选择；系统默认禁止选择，如input框不加入该属性会导致无法选中。

    1. 主框架已加载，无需再次引入。
    2. 此类为指令，直接在html中使用，restrict: 'A'。
       HTML:
   <input type="text" i-dom-select />
