    根据消息框id，删除消息框

    Parameters:

    nametypeflagdescription
    id String

    生成消息框时的id，如生成时没有配置id参数，则无法通过该方法删除消息框。

    Example:

     // 注入iMessage服务。
 app.controller('***Controller', ['iMessage', function(iMessage) {
     // 显示一个全局的消息提醒。
     iMessage.show({
         id: 'test',
         title: '标题',
         content: '内容'
     });

     // 通过代码删除消息提醒框。
     iMessage.remove('test');
 }]);

    显示消息提示框。

    Parameters:

    nametypeflagdescription
    json JSON Object 
     json: {
     id: '',                     // 消息框id，默认可忽略，带有id参数的，可通过调用remove(id)方法去除消息显示。
     level: 1,                   // 1->成功;2->消息；3->警告；4->异常。默认为2
     title: '标题',              // *必填项
     time: '时间',               // 默认取当前客户端时间。
     timeout: '超时消失',        // 默认5秒（int类型）, '0'不自动消失（注意：'0'必须传递字符串）。
     content: '内容',           // *必填项
     drag: false,               // 是否允许拖动，true为允许拖动，默认不允许。
     opacity: 1,                // 背景透明度，默认为1，既不透明。
     fn: 'fn'                   // 点击消息框时响应的方法：callScope.fn，在json中传入方法名后还需传入callScope参数值：fn所属域对象
 }
    checkSame Boolean

    默认false，当输入true时，自动过滤重复项，重复项指与上一次一样既过滤不显示。

    callScope Object

    方法触发对象，用于点击事件时调用。

    scope Object

    作用域，用于content内容为html时，触发与作用域相关的方法。

    Example:

     // 注入iMessage服务。
 app.controller('***Controller', ['iMessage', function(iMessage) {
     // 显示一个全局的消息提醒。
     iMessage.show({
         title: '标题',
         content: '内容'
     });
 }]);
